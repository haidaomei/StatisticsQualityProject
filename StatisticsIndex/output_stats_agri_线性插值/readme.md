这里我要求对一个前后有数据的闭区间(这个闭区间涵盖了group最大元组数)中间进行线性插值 比如0 0 10 10 101 101 10 0 0 0 0 1 1 0 1 0 0
那么前面俩是00 中间10到倒数第三个1是一个闭区间 后面俩是00 那么那个闭区间就用线性插值了
首位的话就用最小二乘法的单元线性回归线性外推(基于线性插值结果回归) 我们做一个线性补全数据 这部分的数据从aligned里读出来 输出的数据结构也和其相同
基于线性插值结果回归那肯定是包含了我们线性插值之后的数据咯 如果某个group只有1-2个非nan 不是哥们我们之前不是剔除了系统性缺失的吗怎么可能出现这种情况
