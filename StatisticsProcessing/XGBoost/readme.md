农业机械总动力 (万千瓦) 
粮食单位面积产量 (公斤/公顷)

农药使用量 (万吨) 
农村用电量 (亿千瓦小时) 
农用化肥施用折纯量 (万吨) 
有效灌溉面积 (千公顷) 
农作物受灾面积 (千公顷) 

那我们就可以确定我们基本的机器学习模型的因变量和自变量

因变量:单产

自变量:机械化 薄膜 农药 用电 化肥 灌溉 受灾
