什么是双重差分法?先看一个场景 假设河南2020年遭遇大洪涝 旁边山东风调雨顺 我如何去看河南被灾害的净影响
实际上有灾害净影响=(河南2020产量-河南2019产量)-(山东2020产量-山东2019产量)
实际上是这样的 河南自己减自己有2020全国趋势-2019全国趋势+河南自身2020趋势-河南自身2019趋势+灾害冲击
山东自己减自己有2020全国趋势-2019全国趋势+山东自身2020趋势-山东自身2019趋势
二者相减结果就是河南自身趋势变化-山东自身趋势变化+灾害冲击 那么我们尽量要求这个河南和山东自身趋势变化要接近 也就是它们普通时刻(没灾害)各维度都接近吧 地域单产什么的 只要满足这个假设 这个残留项就消失了 我们可以得到干净的灾害冲击净影响
当然我们选用固定效应模型来做did 因为它普通拿数据相减不可信 固定效应模型回归出来还会有p值(表征结果可信度的)

那我们如何理解?我们可以:每省每年单产=每个省固定底子+每年固定底子+爆出来的特殊事件 我们求一下爆出来的特殊事件就能得到特殊事件的灾害冲击情况了

这个公式实际上就长成固定效应模型 我们给每省和每年的固定效应都抽出来

我们回顾一下之前的固定效应模型 我提出了公式Y_单产=黑龙江省固定自变量成分*bool_黑龙江+北京固定自变量成分*bool_北京+广西固定自变量成分*bool_广西+2023固定自变量成分*bool_2023+2022固定自变量成分*bool_2022+2021固定自变量成分*bool_2021+因年省都改变的自变量成分(这部分就是f(灾害)、g(机械化)、h(土地面积)等等)

总公式是Y = (∑ 省份布尔 × 省份固定常数) + (∑ 年份布尔 × 年份固定常数) + f(灾害) + g(机械化)

我们改写它 改写成

Y = (∑ 省份布尔 × 省份固定常数) + (∑ 年份布尔 × 年份固定常数) + f(特定受灾省份年份bool) + g(变化的变量的集合 如机械化+薄膜等)

我们的核心是 你放入什么东西 那么系数就表征什么 这就是回归的一个大概

直观来看 后面的f(特定受灾年份bool)是f(灾害)的一个子集 那其他跑去哪了 答案是可能会被误差项或者固定项吸收掉吧

我们的Y = (∑ 省份布尔 × 省份固定常数) + (∑ 年份布尔 × 年份固定常数) + f(特定受灾省份年份bool) + g(变化的变量的集合 如机械化+薄膜等)就是双重差分法的一个公式 我们看f 能看出特定年份特定受灾省份的这个灾害情况

当然 这本身就偏离了灾害净影响=(河南2020产量-河南2019产量)-(山东2020产量-山东2019产量)这个公式 前面方便理解而已
